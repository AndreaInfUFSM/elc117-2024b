## Sobre a disciplina

 - Professora: Andrea Schwertner Charão (andrea@inf.ufsm.br)  
 - Horários: terças e quintas, das 10:30 às 12:30, salas 260 (Anexo A - CT) e 334 (Prédio 07 - CT)
